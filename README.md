# Ted's Voice Academy Website

Voice coaching for singers, speakers, and ensembles. This is the official website for Ted's Voice Academy, serving clients in Lacey, WA and virtually worldwide.

## Tech Stack

- **Framework:** [Astro](https://astro.build)
- **Hosting:** [Netlify](https://netlify.com)
- **Forms:** Netlify Forms
- **CMS:** Decap CMS (planned)

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
/
├── public/              # Static assets (images, favicon, etc.)
├── src/
│   ├── assets/          # Processed assets
│   ├── components/      # Reusable components
│   ├── layouts/         # Page layouts
│   ├── pages/           # Page routes
│   └── styles/          # Global styles & design tokens
├── astro.config.mjs     # Astro configuration
├── netlify.toml         # Netlify configuration
└── package.json
```

## Pages

- `/pass-profile` - PASS Profile assessment

- `/pricing` - Pricing & packages
- `/blog` - Blog
- `/workshops` - Workshops & presentations
- `/vocal-health` - Vocal Health Hub

## Design System

Three visual identities within one cohesive site:

1. **TVA Parent:** Deep navy (#0a1628) + bright gold (#d4a84b)
2. **AVF Section:** Rich teal (#1a6b7c) + amber gold (#d4a039)
3. **PASS Section:** Deep teal (#1a4a5c) + bright green (#3db58c)
